請將maker解壓縮至當前目錄
會出現下列兩個資料夾

TraningData
=============================================================
1. 將所需要處理的音檔16K(.wav)放進 wav資料夾
2. 將所需要處理的原文本UTF8(.txt)放進  original_word資料夾
3. 將所需要處理的注音文本UTF8(.txt)放進 word資料夾
4. 點開make資料夾中的  make.bat 更改路徑並執行處理程式
5. 將full mono raw資料夾中的資料取出即可丟進HTS訓練
=============================================================

Demo
=============================================================
1. 將HTS所訓練出來的模型(.htsvoice)放進 htsvoice資料夾
2. 將所需要合成的的文本UTF8(.txt)放進  worddata資料夾
3. 點開make資料夾中的  make.bat 更改路徑與模型並執行處理程式
4. 將wav資料夾中的資料取出即為所得
=============================================================
